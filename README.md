## ✨ Features

- 🎙️ **Voice engine** — define your brand personality once, AI writes in your voice consistently
- 🔄 **Multi-platform output** — LinkedIn, X (Twitter), Instagram, and Threads — each optimized for the format
- ✍️ **Tone controls** — casual, authoritative, inspirational, controversial — dial it in
- 💡 **Content calendar generator** — weekly post plans generated from a single topic
- 🪝 **Hook library** — battle-tested opening formulas that stop the scroll
- 📊 **Engagement prediction** — AI scores posts before you publish them
- ⚡ **One input, 5 platform versions** — paste your idea, get everything

## 🚀 Quick Start

```bash
git clone https://github.com/jayblast-spec/postos.git
cd postos
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
